2Ô∏è‚É£ Results Webhook Listener
Endpoint POST /results

Purpose
Receives webhook event from HyperVerge after a transaction is completed.

Flow
Configure this endpoint as your HV Webhook URL

HV backend will send a POST request when SDK flow finishes

Extract transactionId (trxnId) from webhook payload

Call HV Results (Logs) API using this transaction ID
https://documentation.hyperverge.co/accessing-results/results-api

Store or log the response (recommended)

Respond back to HV with HTTP 200 OK

3Ô∏è‚É£ Outputs Fetch API
Endpoint POST /outputs 

Purpose
Allows frontend to fetch final processed outputs after SDK completion.

Flow
Frontend ‚Üí Backend

Sends transactionId in request body

Backend Steps

Receive transaction ID

Call HV Results (Logs) API

Call HV Outputs API
https://documentation.hyperverge.co/accessing-results/output_api

Return Outputs API response to frontend
